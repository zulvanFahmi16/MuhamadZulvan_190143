    Sistem Persamaan Aljabar Linier¶

    Jika terdapat Sistem Persamaan Linier dengan n buah variabel bebas

    \[ \begin{array}{l} \boldsymbol{a}_{11} \boldsymbol{x}_{1}+\boldsymbol{a}_{12} \boldsymbol{x}_{2}+\boldsymbol{a}_{13} \boldsymbol{x}_{3}+\ldots+\boldsymbol{a}_{1 n} \boldsymbol{x}_{\boldsymbol{n}}=\boldsymbol{C}_{1} \\ \boldsymbol{a}_{21} \boldsymbol{x}_{1}+\boldsymbol{a}_{22} \boldsymbol{x}_{2}+\boldsymbol{a}_{23} \boldsymbol{x}_{3}+\ldots+\boldsymbol{a}_{2 n} \boldsymbol{x}_{n}=\boldsymbol{C}_{2} \\ \vdots \\ \boldsymbol{a}_{\boldsymbol{m} 1} \boldsymbol{x}_{1}+\boldsymbol{a}_{\boldsymbol{m} 2} \boldsymbol{x}_{2}+\boldsymbol{a}_{m 3} \boldsymbol{x}_{3}+\ldots+\boldsymbol{a}_{\boldsymbol{m}} \boldsymbol{x}_{\boldsymbol{n}}=\boldsymbol{C}_{\boldsymbol{n}} \\ \text { Matriks: }\left[\begin{array}{cccc} \boldsymbol{a}_{11} & \boldsymbol{a}_{12} & \ldots & \boldsymbol{a}_{1 n} \\ \boldsymbol{a}_{21} & \boldsymbol{a}_{22} & \ldots & \boldsymbol{a}_{2 n} \\ \boldsymbol{a}_{31} & \boldsymbol{a}_{32} & \ldots & \boldsymbol{a}_{3 n} \\ \vdots & \vdots & \cdots & \vdots \\ \boldsymbol{a}_{n 1} & \boldsymbol{a}_{n 2} & \ldots & \boldsymbol{a}_{n n} \end{array}\right]\left[\begin{array}{c} \boldsymbol{x}_{1} \\ \boldsymbol{x}_{2} \\ \boldsymbol{x}_{3} \\ \vdots \\ \boldsymbol{x}_{n} \end{array}\right]=\left[\begin{array}{c} \boldsymbol{C}_{1} \\ \boldsymbol{C}_{2} \\ \boldsymbol{C}_{3} \\ \vdots \\ \boldsymbol{C}_{n} \end{array}\right] \end{array} \]

    Bisa diselesaikan dengan :

    • Algoritma gauss naif
    • Algoritma gauss jordan
    • Algoritma gauss seidel

    Algoritma Gauss Naif¶

    1. Membagi persamaan pertama dengan koefisien \(a11\). (normalisasi) Tujuan normalisasi adalah agar koefisien \(x1\) menjadi \(1\).
    2. Kalikan persamaan yang sudah dinormalisasi (persamaan pertama) dengan \(a21\).
    3. Mengurangibaris kedua dan baris ketiga dengan baris pertama.
    4. Kalikan persamaan pertama (sudah dinormalisasi) dengan koefisien tertentu hingga \(a11 =a31\).
    5. Kurangkan persamaan ketiga dengan hasil dari langkah 4.
    6. Baris kedua dibagi koefisien \(a22\). (NORMALISASI untuk persamaan kedua).Tujuannya agar koefisien $x2 $menjadi 1
    7. Kalikan persamaan kedua dengan suatu koefisien sehingga \(a22 = a32\).
    8. Kurangkan persamaan ketiga dengan persamaan kedua darilangkah ke-7.

    Algortima gauss jordan¶

    Dengan metode Gauss Jordan matriks A diubah sampai terbentuk identitas:

    \({[\boldsymbol{A} \mid \boldsymbol{I}] \boldsymbol{X}=\boldsymbol{C}}\)

    diubah menjadi

    \({\left[\boldsymbol{I} \mid \boldsymbol{A}^{-1}\right] \boldsymbol{X}=\boldsymbol{C}^{*}}\)

    merupakan matriks C yang mengalami beberapa transformasi, sehingga :

    Code Gauss Jordan¶

    # Import NumPy
import numpy as np
import sys

# input angka
n = int(input('Masukkan angka : '))

# membuat array
a = np.zeros((n,n+1))
x = np.zeros(n)

# masukkan koefisien nilai matrix
for i in range(n):
    for j in range(n+1):
        a[i][j] = float(input( 'a['+str(i)+']['+ str(j)+']='))

# Gauss Jordan
for i in range(n):
    if a[i][i] == 0.0:
        sys.exit('pembagi nol -- code gagal')

    for j in range(n):
        if i != j:
            ratio = a[j][i]/a[i][i]

            for k in range(n+1):
                a[j][k] = a[j][k] - ratio * a[i][k]

# mengolah solusi

for i in range(n):
    x[i] = a[i][n]/a[i][i]

# menampilkan solusi
print('\nSolusi : ')
for i in range(n):
    print('X%d = %0.2f' %(i,x[i]), end = '\t')

    Algoritma Gauss Seidel¶

    Sering digunakan pada persamaan yang berjumlah besar.

    Dilakukan dengan iterasi yang memberi nilai awal untuk \(x1 = x2 = x3 =... = xn = 0\).

    berlainan dengan metode Gauss Jordan dan Gauss Naif , metode ini menggunakan iterasi dalam menentukan nilai \(x1 , x2 , x3 , ..., xn\)

    Algoritmanya

    1. Berinilai awal \(x1 = x2 = x3= ... = xn = 0\)
    2. Hitung

    \(\boldsymbol{x}_{1}=\frac{\boldsymbol{C}_{1}-\left(\boldsymbol{a}_{12} \boldsymbol{x}_{2}+\mathbf{a}_{13} \boldsymbol{x}_{3}+\boldsymbol{a}_{14} \boldsymbol{x}_{4}+\ldots+\boldsymbol{a}_{1 n} \boldsymbol{x}_{n}\right)}{\boldsymbol{a}_{11}}\)

    Karena \(x2 = x3 = x4 = ... = xn = 0​\), maka :

    \(x_{1}=\dfrac{c_{1}}{a_{11}}\)

    1. \(x1\) baru yang diperoleh dari langkah 2 digunakan untuk menentukan \(x2\) .

    Baris 2 : \(a21x1 + a22x2 + a23x3 + ... + a2nxn = C2\)

    ​

    1. Menghitung x3

    Baris 3 : \(a31x1 + a32x2 + a33x3 + ... + a3nxn = C3\)

    \(a33x3 = C3 – a31x1 – a32x2 – … – a3nxn\)

    \(x_{3}=\frac{C_{3}-a_{31} x_{1}-a_{32} x_{2}-a_{34} x_{4}-\ldots-a_{3 n} x_{n}}{a_{33}}\) \(x_{3}=\frac{C_{3}-a_{31} x_{1}-a_{32} x_{2}}{a_{33}}\)

    ​

    1. Cara ini diulangi terus menerus sampai \(xn\)

    2. Lakukan iterasi ke-2 untuk menghitung \(x1 , x2 , x3, ..., xn\) baru

    \(x_{1}=\frac{\boldsymbol{C}_{1}-\mathbf{a}_{12} \boldsymbol{x}_{2}-\mathbf{a}_{13} \boldsymbol{x}_{3}-\ldots-\mathbf{a}_{1 n} \boldsymbol{x}_{n}}{\mathbf{a}_{11}}\) \(\boldsymbol{x}_{2}=\frac{\boldsymbol{C}_{2}-\boldsymbol{a}_{21} \boldsymbol{x}_{1}-\mathbf{a}_{23} \boldsymbol{x}_{3}-\ldots-\mathbf{a}_{2 n} \boldsymbol{x}_{n}}{\mathbf{a}_{22}}\) \(\vdots\) \(\boldsymbol{x}_{n}=\frac{\boldsymbol{C}_{n}-\boldsymbol{a}_{11} \boldsymbol{x}_{1}-\mathbf{a}_{12} \boldsymbol{x}_{2}-\mathbf{a}_{13} \boldsymbol{x}_{3}-\ldots-\mathbf{a}_{1(\boldsymbol{n}-1)} \boldsymbol{x}_{n-1}}{\boldsymbol{a}_{n n}}\)

    1. Mencari kesalahan perulangan \(|\varepsilon a|\) dengan cara :

    \(\boldsymbol{x}_{i} \Rightarrow\left|\varepsilon_{\boldsymbol{a}}\right|=\left|\frac{\boldsymbol{x}_{i(\text { baru })}-\boldsymbol{X}_{i(\text { lama })}}{\boldsymbol{x}_{i(\text { baru })}}\right| * 100 \%\) \(\vdots\) \(\boldsymbol{x}_{n} \Rightarrow\left|\varepsilon_{\boldsymbol{a}}\right|=\left|\frac{\boldsymbol{x}_{\boldsymbol{n}(\text { baru })}-\boldsymbol{X}_{\boldsymbol{n}(\text { lama })}}{\boldsymbol{x}_{\boldsymbol{n}(\text { baru })}}\right| * 100 \%\)

    1. Perulangan diteruskan hingga didapat \(|\varepsilon a |< |\varepsilon s|\)

    Code Gauss Seidel¶

    def seidel(a, x ,b):    
    n = len(a)              
    for j in range(0, n):       
        d = b[j]                
        for i in range(0, n):   
            if(j != i):
                d-=a[j][i] * x[i]       
        x[j] = d / a[j][j]      
    return x    

n = 3                           
a = []                          
b = []      

x = [0, 0, 0]                       
a = [[4, 1, 2],[3, 5, 1],[1, 1, 3]]
b = [4,7,3]
print(x)

for i in range(0, 25):          
    x = seidel(a, x, b)
    print(x)
